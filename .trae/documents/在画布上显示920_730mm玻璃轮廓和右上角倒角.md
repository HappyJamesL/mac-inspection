## 在画布上显示920*730mm玻璃轮廓和右上角倒角

### 问题分析
当前CanvasStage组件没有绘制完整的玻璃轮廓，只绘制了内部的panels和mask区域。需要添加玻璃外部轮廓的绘制，包括右上角20*10mm的倒角。

### 解决方案
修改`d:\DEV\Code\JamesRepo\mac-insp\mac-inspection\frontend\src\components\CanvasStage.vue`文件，在`drawBackground`函数中添加玻璃轮廓的绘制逻辑。

### 具体修改点

1. **在`drawBackground`函数中添加玻璃轮廓绘制**：
   - 定义带有右上角20*10mm倒角的矩形路径
   - 使用Konva.Line或Konva.Path绘制轮廓
   - 设置合适的样式（颜色、线宽等）
   - 确保轮廓绘制在panels和mask之前

2. **玻璃尺寸配置**：
   - 当前默认值已为920000um x 730000um（即920mm x 730mm），无需修改
   - 确保使用正确的单位转换（um -> mm）

3. **倒角实现**：
   - 右上角20mm（X方向）x 10mm（Y方向）的倒角
   - 使用路径点定义：[x1,y1, x2,y1, x2,y2, x3,y3, x4,y3, x4,y4, x1,y4, x1,y1]
   - 其中x3 = x2 - 20mm，y3 = y2 + 10mm

4. **坐标转换**：
   - 使用现有的`toScreen`和`toScreenY`函数进行坐标转换
   - 玻璃中心对齐画布中心

### 修改策略
- 在`drawBackground`函数开始处绘制玻璃轮廓
- 使用白色边框，宽度2px，与黑色背景形成对比
- 确保轮廓在所有其他元素之下

### 预期效果
- 画布上显示完整的920*730mm玻璃轮廓
- 右上角有20*10mm的倒角
- 轮廓清晰可见，与内部元素层次分明
- 适配不同的缩放比例

### 关键代码实现
```javascript
// 玻璃轮廓绘制逻辑
const glassW = glassConfig.size.w;
const glassH = glassConfig.size.h;
const chamferW = 20000; // 20mm = 20000um
const chamferH = 10000; // 10mm = 10000um

// 定义玻璃轮廓路径点（带有右上角倒角）
const glassPath = [
  -glassW/2, glassH/2, // 左下角
  glassW/2, glassH/2,  // 右下角
  glassW/2, -glassH/2 + chamferH, // 右下角上移chamferH
  glassW/2 - chamferW, -glassH/2, // 右移chamferW，左上角
  -glassW/2, -glassH/2, // 左上角
  -glassW/2, glassH/2   // 回到左下角
];

// 转换为屏幕坐标
const screenPath = glassPath.flatMap((val, idx) => 
  idx % 2 === 0 ? toScreen(val) : toScreenY(val)
);

// 绘制玻璃轮廓
mainLayer.add(new Konva.Line({
  points: screenPath,
  stroke: '#ffffff',
  strokeWidth: 2,
  closed: true,
  listening: false,
  opacity: 0.8
}));
```
